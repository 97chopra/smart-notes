# 🧠 Smart Notes

A modern, AI-powered notes app built with HTML, CSS and Vanilla JS.

## Features
- Create, edit, delete notes
- Markdown support with live preview
- Tag system with sidebar filter
- Live search and sort
- Pin notes and colour labels
- Export notes as .md or .txt
- AI summarise, tag suggest, and rewrite

## Tech Stack
- HTML / CSS / Vanilla JS
- [marked.js](https://marked.js.org/) — Markdown rendering
- OpenAI API — AI features

## Setup
1. Clone the repo
2. Add your OpenAI API key in `ai.js`
3. Open `index.html` in your browser — no build step needed!
